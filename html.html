¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para la creación y estructuración de páginas web. No es un lenguaje de programación, sino un lenguaje que define la estructura del contenido mediante etiquetas.

Historia del HTML

HTML fue creado por Tim Berners-Lee en 1991 como una forma de compartir documentos en la World Wide Web (WWW). Desde entonces, ha evolucionado a través de varias versiones:

  • HTML 1.0 (1993): Primera versión oficial.
  • HTML 2.0 (1995): Estableció estándares básicos.
  • HTML 3.2 (1997): Introdujo tablas y mejoras en diseño.
  • HTML 4.01 (1999): Incorporó hojas de estilo en cascada (CSS).
  • XHTML (2000): Versión más estricta basada en XML.
  • HTML5 (2014 - Actualidad): Añadió soporte para video, audio y elementos interactivos.

¿Para qué sirve HTML?

HTML se utiliza para:

  • Crear la estructura de páginas web.
  • Integrar imágenes, videos y enlaces.
  • Organizar el contenido con encabezados, listas y párrafos.
  • Definir formularios para recopilar información del usuario.

¿Cómo funciona HTML?

HTML trabaja con etiquetas que definen el contenido de una página. Ejemplo básico:

        
<!DOCTYPE html>
<html>
    <head>
        <title>Mi primera página</title>
    </head>
    <body>
        <h1>Hola, mundo</h1>
        <p>Bienvenido a mi página web</p>
    </body>
</html>

Cada etiqueta cumple una función específica. Por ejemplo:

  • <h1> a <h6>: Encabezados.
  • <p>: Párrafos.
  • <a>: Enlaces.
  • <img>: Imágenes.
  • <table>: Tablas.

¿Qué se necesita para programar en HTML?

Para trabajar con HTML, se requieren:

  1. Editor de texto: Bloc de notas, VS Code, Sublime Text, etc.
  2. Navegador web: Chrome, Firefox, Edge, Safari.
  3. Conocimientos básicos de etiquetas HTML y su sintaxis.

Lenguajes que complementan HTML

HTML se combina con otros lenguajes para mejorar su funcionalidad:

  • CSS (Cascading Style Sheets): Se usa para dar estilo y diseño a las páginas web.
  • JavaScript: Permite agregar interactividad y dinamismo.
  • PHP y Python: Se usan en la programación del lado del servidor.
  • SQL: Para gestionar bases de datos.

Conclusión

HTML es el pilar fundamental del desarrollo web. Su combinación con CSS y JavaScript permite la creación de sitios web dinámicos y atractivos. Conocer HTML es el primer paso para aprender desarrollo web.
